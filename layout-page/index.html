What is Layouts?

Layout is the way text, images and other content is placed and arranged on webpage.

There are three different layout methods.

  1. Float Layout: The old way of building layouts of all sizes, using the float CSS property float sill is being used but getting outdated fast.
  2. Flexbox: Modern way of laying out elements in a 1D row without using floats, perfect for component layouts.
  3. CSS grid: For laying out element in a fully-fledged 2D grid. perfect for page layouts and complex components

Float Layour

Float element are removed from the normal flow: "out of flow" similar to what happend when a element change from relative position to absolute position

But Text and inline elements will still wrap around the floated element

The container will not adjust its height to the float elements and will collapsed(height = 0) if all elements are floats within the container.
