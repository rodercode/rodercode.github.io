The Basic of CSS

posted by Marcus Groth on Friday, March 31st 2023

What is CSS

  • CSS is standing for: Cascading Style Sheets
  • CSS describes the visual style of the content written in HTML
  • CSS consists of countless properties that developers use to
    format the content: properties about font, text, spacing, layout,
    etc. basically properties are used to style a element

How we select and style elements

  • Element we select to style is called the selector
  • Text-align, font-size and color are properties and we are assigne a value to them to style them.
  • Properties and value together are called style or declaration
  • The whole block within the selector is called declaration block

The Three type of consist

Inline CSS

  • Inline CSS is when you write css code inside a element
  • Inline CSS is usually very bad practice so it shouldn't be use

Internal CSS

  • Internal CSS is using a style element to style html content
  • Style element is located inside the head element not the body element
  • internal CSS is not optimal when dealing with huge amount of CSS code

External CSS

  • External CSS is when you working in a separate file a CSS file
  • we use a link element to connect html file to a css file

Id selector

This is an id selector we can add an id to our html element and then use #id-name
to only style that specific element. but we can only assign one element to an id and id selector doesn't use as often as class selector cause class selector is well suited for real time projects where we might want to use same identifier for multiple elements.

Class selector

Class selector is similar to how id selector works only difference is that we can select
multiple element and style them

Pseudo classes

Pseudo-classe are used to add styles to element in different state.
for example nav element can have four different state

  • linked(display hyperlink)
  • visited(hyperlink you already visited)
  • hover(when you hover your mouse over a link)
  • active(while click on a link)

You can see it as an whole animation taking place from you see
a link until you have clicked on it.
